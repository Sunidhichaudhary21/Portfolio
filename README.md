# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **React.js** and **Tailwind CSS** to showcase my projects, technical skills, and experience.

---

## 🚀 Live Demo

🔗 https://sunidhi11.vercel.app/

---

## 📌 Features

* 🎨 Clean and modern UI/UX design
* 📱 Fully responsive across all devices
* ⚡ Fast performance with Vite
* 🧩 Reusable React components
* 📊 Project showcase section
* 🛠 Skills and technologies display
* 📬 Contact section for easy communication

---

## 🛠 Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Build Tool:** Vite
* **Version Control:** Git & GitHub
* **Deployment:** Vercel

## ⚙️ Installation & Setup

Clone the repository and run locally:

```bash
git clone https://github.com/Sunidhichaudhary21/Portfolio.git
cd Portfolio
npm install
npm run dev
```
